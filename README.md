# Luka Demo

Demo funcional (con datos simulados) inspirada en una app fintech de finanzas y pagos.

## Stack
- Next.js (App Router) + TypeScript
- Tailwind CSS (modo oscuro)
- Recharts (gráficos)
- Framer Motion (animaciones)
- Datos mock con persistencia en localStorage

## Ejecutar en local
1) Instalar dependencias:
   npm install
2) Levantar en desarrollo:
   npm run dev
3) Abrir:
   http://localhost:3000
